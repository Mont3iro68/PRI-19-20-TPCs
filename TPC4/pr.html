ABSTRACT:

O Node.js é um interpertador de JavaScript assíncrono com código Open-Source.

É utilizado para correr Scripts do lado do servidor, permitindo a produção de páginas web dinâmicas.

Neste trabalho foi desenvolvido um servidor em Node.js capaz de responder a pedidos HTTP do tipo GET para a obtenção de informação sobre uma música através do ficheiro XML da mesma.

Para além disso foi criado um ficheiro XSL transformando os documentos XML para serem apresentados em HTML. Este ficheiro é depois enviado pelo servidor, para que seja carregada uma página contendo a informação presente no documento XML


Deliverables:

  • pr.xml
  • pr.xsl
  • doc2html.xsl
  • servidor.js
