ABSTRACT:

Este trabalho tinha como objetivo a utilização da linguagem XSL (eXtensible Stylesheet Language). O XSL é uma linguagem de estilo que define como devem ser apresentados os ficheiros XML de um tipo.

É utilizado o XSLT para fazer transformações nos ficheiros, para por exemplo HTML.

Neste trabalho foi desenvolvido um ficheiro XSL para a produção deste project record como também outro ficheiro do mesmo tipo para a produção de um website.
